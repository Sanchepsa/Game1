# Game1

ТЗ:
Сделать стрелялку:
1) Спрайт главного героя
2) Спрайты врагов
3) Карта (пока что не реализована, есть лишь небольшая площадка)
4) Оружие
5) Реализация механики движения, стрельбы, начисления очков и т.д.

Игра написана на языке Python, используемая библиотека — PyCharm.
Вскоре проект будет дополнен некоторыми элементами.

Игра "Cheeshoota"

Правила игры:
Нажимаем кнопку "PLAY".
Мы играем за сыр, управляем его движением с помощью кнопок "вправо", "влево".
Стрелять нужно по мышам, которые появляются справа и слева.
Чтобы стрелять, нажимаем "F".
Прикосновение к мышам отнимает здоровье.
Убийство одной мыши — одно очко.
Игра заканчивается, если шкала здоровья стала полностью красной.
